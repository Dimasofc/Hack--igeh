## ❯ Instalasi
Salin semuanya paste di termux terus enter!!!
```bash
apt update;apt upgrade
apt-get install python -y
apt-get install git -y
git clone https://github.com/Din-zUgex95/nghek-igeh
cd nghek-igeh
pip install -r requirements.txt
chmod 777 run
./run
```
